■ビルド方法
  $ ./autogen.sh
  $ make

■起動方法
  test_player [options] /path/to/media

  アプリケーションのオプション:
    -a, --audio                  音声ストリームを有効にする
    -v, --video                  映像ストリームを有効にする
    -q, --queue                  HW デコーダの前段に queue を挿入する
    -s, --seek-interval=INTERVAL seek 間隔(msec) デフォルトは 5000

■操作方法
  操作コマンドと対応するキーは以下のとおりです。

  コマンド         キー      説明
  終了             [q]       アプリケーションを終了します
  一時停止・再開   [SPACE]   一時停止・再開のトグル
  停止・再開       [s]       再生停止・初めから再生再開のトグル
  Seek（前方）     [f]       前方に Seek します
  Seek（後方）     [b]       後方に Seek します
  Seek（先頭）     [r]       先頭に戻ります
  Trick Play       [1]       1.5倍速早送り再生
  Trick Play       [2]       2倍速早送り再生
  Trick Play       [4]       等倍速巻き戻し再生
  Trick Play       [5]       1.5倍速巻き戻し再生
  Trick Play       [6]       2倍速巻き戻し再生

■プロパティの設定
・プロパティ設定ファイル
  カレントディレクトリに properties.conf を置くことにより、各エレメント
  のプロパティを設定することができます。properties.conf は以下のフォーマッ
  トで記述してください。

  例）
  ----------------------------------------------------------------------
    [エレメント名]
    プロパティ名1 = 設定値1
    プロパティ名2 = 設定値2

    [rtofbdevsink]
    device = "/dev/fb0"
    use-dmabuf = TRUE

    [pre-adec-queue]
    max-size-buffers = (uint)300
   ---------------------------------------------------------------------

・エレメント名について
  エレメント名は gst_element_register() で登録した名前で指定します。

・型について
  対応しているプロパティの型は以下のとおりです。

  |---------+------------------+-------------------------------+---------------------------------------|
  | 型      | 設定内容         | 設定例                        | 備考                                  |
  |---------+------------------+-------------------------------+---------------------------------------|
  | boolean | true/false       | use-dmabuf = TRUE             | true/false は大文字小文字の区別なし   |
  |---------+------------------+-------------------------------+---------------------------------------|
  | int     | 32bit 符号あり   | some-property = 123           | 型を指定しない場合は int として解釈   |
  |         |                  | some-property = (int)123      |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | uint    | 32bit 符号なし   | some-property = (uint)123     |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | int64   | 64bit 符号あり   | some-property = (int64)123    |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | uint64  | 64bit 符号なし   | some-property = (uint64)123   |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | float   | 単精度浮動小数点 | some-property = 0.123         | 型を指定しない場合は float として解釈 |
  |         |                  | some-property = (float)0.123  |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | double  | 倍精度浮動小数点 | some-property = (double)0.123 |                                       |
  |---------+------------------+-------------------------------+---------------------------------------|
  | string  | 文字列           | device = /dev/fb0             | 上記以外はすべて文字列として解釈      |
  |         |                  |                               | 空白を含む場合は「"」で囲む           |
  |---------+------------------+-------------------------------+---------------------------------------|

・注意事項
  properties.conf でエレメント filesrc の location を設定した場合、引数
  で設定されたパスが上書きされます（properties.conf の内容が優先されます）。
